함수연습

    0         1         2
    012345678901234567890123456789
    ABCDEFGHIJKLMNOPQRSTUVWXYZ

legth : 객체(문자열)의 길이

slice() : 지정된 문자열을 추출해준다
(a,b) a부터 b-1까지 - 지정된 문자 앞에 까지
(-10) 뒤에서부터 10개 문자를 추출한다
(-10,-5) 뒤에서부터 10개번쨰부터 뒤쪽 -5앞까지 문자를 추출한다

substring(a,b) : 지정된 문자열(a부터 b앞까지) slice와 비슷

substr(-a) :뒤에서부터 a번째 까지 나온다 substring과 substr을 합친것이 slice

replace(a,b) : a를 b로 치환

toUpperCase() : 대문자로 치환

toLowerCase() : 소문자로 치환

concat(a,b) : a 와 b를 결합한다.

trim() : 양쪽 공백을 삭제한다.

trimStart() : 앞쪽 공백을 삭제한다.

trimEnd() : 뒤쪽 공백을 삭제한다.

replaceAll(" ","") : 모든 공백을 삭제한다.

charAt() : 자바에서 문자한개를 가져온다. 변수[0]도 똑같음
charCodeAt() : 아스키 코드로 가져옴

split() : 구분자를 기준으로 잘라준다. 잘라진 각각의 요소는 배열로 저장된다.
지정된문자에 ','구분 한다.

indexOf() : 특정문자의 위치값을 찾아준다.

lastIndexOf() : 특정문자의 위치값을 뒤에서 부터 찾아준다.

search() : 특정문자의 위치값을 찾아준다.

match() :찾는값이 있으면 '문자열' 을 반환, 없으면 '공백'을 변환한다

includes() : 찾는문자열을 포함하고있으면 true 없으면 false를 반환한다.
