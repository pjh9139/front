Dispaly 속성

Dispaly속성은 각각의 요소를 어떻게 보여줄지를 결정한다.

속성처리값? visibility(기본값:보이기)/ hidden(감추기)

/ inherit(부모요소의 값만 보여줌)

/ none : 보여주지 않는다

/ inline : 인라인상태 (자신의 고유한 영역만 확보) - 단어의 의미로 해석? 크기조정불가

/ block : 박스영역을 차지하는 속성 - 문단의 의미로 해석? 크기조정가능

/ inline-block : inline의 속성과 block의 속성을 합성
